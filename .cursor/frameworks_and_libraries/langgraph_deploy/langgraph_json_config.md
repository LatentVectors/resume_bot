# LangGraph CLI and Configuration

This document provides a comprehensive reference for the `langgraph.json` configuration file and the `langgraph` command-line interface.

## Configuration File (`langgraph.json`)

This file defines the structure, dependencies, and behavior of a deployable LangGraph agent.

| Key                             | Type                         | Description                                                                                                                                                                                                     |
| :------------------------------ | :--------------------------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `dependencies`                  | `array[string]`              | **Required.** An array of dependencies. Can be a PyPI package name, or a local directory path containing a `pyproject.toml`, `setup.py`, or `requirements.txt`. Use `"."` to include the current local package. |
| `graphs`                        | `object`                     | **Required.** A mapping from a public graph ID to the path of the graph object. Path format: `./path/to/file.py:variable_name`. The variable can be a compiled graph or a function that returns a graph.        |
| `auth`                          | `object`                     | Authentication configuration. Specifies the path to an `Auth` instance. See authentication guides for details.                                                                                                  |
| `base_image`                    | `string`                     | Optional. Base Docker image for the server (e.g., `langchain/langgraph-server:0.2`). Defaults to `langchain/langgraph-api`.                                                                                     |
| `image_distro`                  | `string`                     | Optional. Linux distribution for the base image. Can be `"debian"`, `"wolfi"`, `"bookworm"`, or `"bullseye"`. Defaults to `"debian"`.                                                                           |
| `env`                           | `string` or `object`         | Path to a `.env` file or a direct mapping of environment variable key-value pairs.                                                                                                                              |
| `store`                         | `object`                     | Configuration for the `BaseStore`, including semantic search and Time-to-Live (TTL).                                                                                                                            |
| `store.index`                   | `object`                     | Optional. Enables semantic search. Contains `embed` (model name or path to function), `dims` (integer), and `fields` (array of JSON paths to embed).                                                            |
| `store.ttl`                     | `object`                     | Optional. Configures item expiration. Fields: `refresh_on_read` (boolean, default `true`), `default_ttl` (float, minutes), `sweep_interval_minutes` (integer).                                                  |
| `ui`                            | `object`                     | Optional. Named definitions of UI components emitted by the agent, pointing to JS/TS files.                                                                                                                     |
| `python_version`                | `string`                     | Optional. Python version to use. Can be `"3.11"`, `"3.12"`, or `"3.13"`. Defaults to `"3.11"`.                                                                                                                  |
| `node_version`                  | `string`                     | Optional. Specify a Node.js version (e.g., `"20"`) to use LangGraph.js.                                                                                                                                         |
| `pip_config_file`               | `string`                     | Optional. Path to a pip config file.                                                                                                                                                                            |
| `pip_installer`                 | `string`                     | Optional. Can be `"auto"`, `"pip"`, or `"uv"`. Defaults to `"auto"` (which uses `uv`).                                                                                                                          |
| `keep_pkg_tools`                | `boolean` or `array[string]` | Optional. If `true`, keeps `pip`, `setuptools`, and `wheel` in the final image. If an array, specifies which of the three to keep. Defaults to `false` (uninstall all).                                         |
| `dockerfile_lines`              | `array[string]`              | Optional. An array of additional lines to append to the generated Dockerfile.                                                                                                                                   |
| `checkpointer`                  | `object`                     | Configuration for the checkpointer.                                                                                                                                                                             |
| `checkpointer.ttl`              | `object`                     | Optional. Configures checkpoint expiry. Fields: `strategy`, `sweep_interval_minutes`, `default_ttl`.                                                                                                            |
| `checkpointer.serde`            | `object`                     | Optional. Tunes deserialization. Fields: `allowed_json_modules` (allow-list of custom Python objects) and `pickle_fallback` (boolean).                                                                          |
| `http`                          | `object`                     | Configuration for the underlying HTTP server (Starlette/FastAPI).                                                                                                                                               |
| `http.app`                      | `string`                     | Optional. Path to a custom Starlette/FastAPI app for adding custom routes.                                                                                                                                      |
| `http.cors`                     | `object`                     | Optional. Defines CORS settings like `allow_origins`, `allow_methods`, etc.                                                                                                                                     |
| `http.configurable_headers`     | `object`                     | Optional. Defines which request headers are exposed as configurable values via `includes`/`excludes` patterns.                                                                                                  |
| `http.logging_headers`          | `object`                     | Optional. Same as `configurable_headers` but for excluding sensitive headers from logs.                                                                                                                         |
| `http.middleware_order`         | `string`                     | Optional. `"auth_first"` or `"middleware_first"` (default).                                                                                                                                                     |
| `http.enable_custom_route_auth` | `boolean`                    | Optional. If `true`, applies auth checks to custom routes.                                                                                                                                                      |
| `http.disable_*`                | `boolean`                    | Optional. A series of flags (`disable_assistants`, `disable_runs`, etc.) to disable built-in API routes or hooks.                                                                                               |
| `http.mount_prefix`             | `string`                     | Optional. A URL prefix for all mounted routes (e.g., `/my-agent/api`).                                                                                                                                          |
| `api_version`                   | `string`                     | Optional. Pins the LangGraph API server to a specific semantic version (e.g., `"0.3"`). Defaults to the latest stable version.                                                                                  |
